# contributing

thanks for your interest in contributing.

## development

```bash
pnpm install
pnpm run build
pnpm run test
```

## pull requests

pretty standard. create a branch, open a PR. before committing, just make sure you've run `pnpm run lint` and `pnpm run check`. i dont want to add precommit hooks. pls.
