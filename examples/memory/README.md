# memory example

durable streams with in-memory storage. ideal for durable objects where state lives in memory.

## run

```bash
pnpm run dev
```

## test

```bash
pnpm run test
```

runs the [durable-streams conformance suite](https://github.com/durable-streams/durable-streams/tree/main/packages/server-conformance-tests) against `http://localhost:8787`.
